Class SynchronizedBag<E>

  • java.lang.Object
    • org.apache.commons.collections4.collection.SynchronizedCollection<E>
      • org.apache.commons.collections4.bag.SynchronizedBag<E>
  • All Implemented Interfaces:
    Serializable, Iterable<E>, Collection<E>, Bag<E>
    Direct Known Subclasses:
    SynchronizedSortedBag

    public class SynchronizedBag<E>
extends SynchronizedCollection<E>
implements Bag<E>
    Decorates another Bag to synchronize its behaviour for a multi-threaded environment.

    Methods are synchronized, then forwarded to the decorated bag. Iterators must be separately synchronized around the loop.

    This class is Serializable from Commons Collections 3.1.
